# zennodes
Displays info about connected nodes for any bitcoin derived cryptocurrency.
![example pic](http://i.imgur.com/qQAxdJX.png)

### Installation
Put the daemon's rpc info into config.json
Run https://github.com/ZenNodes/znodes
